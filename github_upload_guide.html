GitHub upload steps for Folio and Ink website

Save the downloaded file as exactly:

index.html
1

Create a new repository on GitHub

Go to github.com → click New → name it e.g. book-review-blog → click Create repository

2

Upload the file

On the new repo page → click uploading an existing file → drag your index.html → click Commit changes

3

Enable GitHub Pages (to get a live URL)

Settings → Pages (left sidebar) → Source: Deploy from a branch → Branch: main → folder: / (root) → Save

Your live site will be at:

https://YOUR-USERNAME.github.io/book-review-blog/
